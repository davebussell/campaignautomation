Home/Solutions/Integrations

We sit above your tools. We don't replace them.

Our automation layer works with your existing ad platforms, analytics tools, and CRM — not instead of them. You export the data. We build the workflows on top.

Data sources we accept

  • Google Ads (CSV export)
  • Google Analytics 4 (landing page export)
  • Google Search Console (query/page report)
  • HubSpot CRM (contact/deal export)
  • Meta Ads (campaign report export)

Tools we configure automation for

  • Claude (Anthropic)
  • Cursor
  • n8n
  • Zapier
  • Make
  • Airtable

What we don't connect to (yet)

These are Phase 2. CSV uploads only at this stage.

  • Live OAuth API access
  • Direct ad platform write access
  • Real-time bidding APIs

We deliberately work from exported data rather than live API connections. It keeps your accounts secure, avoids permissions overhead, and means we can analyse your data without needing platform access. Live API integrations are on the roadmap for Phase 2.
