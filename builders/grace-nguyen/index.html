Grace Nguyen

Workflow Integration & Governance Consultant
Victoria, BC · she/her · 6 yrs · Governance & strategy

The integration layer where every automated action leaves a trail

This is an illustrative profile of a role we’re actively filling — a representative archetype, not a current team member. If it describes you, apply for this role.

The profile

Grace Nguyen builds the wiring that makes separate marketing tools behave like one accountable system. Working from Victoria, BC, she spends her days inside Zapier, Make, and n8n, plus the custom API glue that commercial connectors never quite cover. Six years of integration work taught her the same lesson repeatedly: the multi-step automation that breaks at 2 a.m. is almost never the one with a human checkpoint in it.

So she designs the checkpoint in from the start. Before a Zap fires a campaign, a Make scenario pushes a contact, or an n8n flow updates a CRM, there is an approval gate and a log entry recording who or what allowed it. Her builds are bounded on purpose, with clear thresholds for what runs unattended and what waits for a person.

That discipline matters most in connective work, where one silent webhook can quietly corrupt data across five systems. Grace makes the trail visible, so a misfire is caught and reversed instead of compounding.

What this builder owns

Approval-gated multi-tool campaign pipelines

Chains Zapier and Make steps so any outbound send or data push pauses at a defined threshold for human sign-off before it executes.

Logged n8n CRM sync with rollback

Builds self-hosted n8n flows that record every record change to an audit log and can replay or reverse a bad sync cleanly.

Custom API middleware with guardrails

Writes the integration layer between tools that lack native connectors, enforcing rate limits, validation, and full request logging on each call.

Cross-platform error and exception routing

Wires failure handlers across Zapier, Make, and n8n so silent webhook breaks surface as alerts with the full context to debug and retry.
